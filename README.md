# Brewin' And Chewin' Legacy

## Overview

Brewin' And Chewin' Legacy is a Minecraft 1.12.2 port of the original Brewin' And Chewin' mod, rebuilt around Farmer's Delight Legacy.

It brings the cozy little tavern side of Farmer's Delight back to 1.12.2: ferment drinks in kegs, age cheese wheels, pour booze into tankards, set snacks on coasters, and generally make your kitchen feel a lot more lively. If you enjoyed the original Brewin' And Chewin', this port is here to let that same flavor fit into an older modded setup.

## Required Dependencies

- Farmer's Delight Legacy

## Mod Compatibility

- Future MC: Uses its sweet berries, honey bottles, honeycombs, wither roses, and some modern Nether or aquatic ingredients when available. If it is not installed, this mod enables local fallback items and recipes where needed.

- Unseen's Dungeon Additions: Provides the preferred glow berries for recipes when available.

- Depths Update: Uses its glow berries when Unseen's Dungeon Additions is not providing them. If neither mod provides glow berries, this mod registers local glow berries with an alternate acquisition path.

- Oceanic Expanse: Uses its kelp, dried kelp, sea pickles, seagrass, glow ink sacs, turtle eggs, and other aquatic ingredients when available. If it is not installed, this mod enables local fallback items, fishing drops, and villager trade sources.

- Netherized / NB: Uses available modern Nether ingredients such as crimson fungus, warped fungus, and shroomlights when available. If they are not installed, this mod uses local fallbacks or approved substitute ingredients.

- Deeper Depths: Dread Nog and related recipes prefer its ominous bottle and Bad Omen effect when available.

- Raids Backport: Provides an alternate ominous bottle and Bad Omen compatibility path when Deeper Depths is not providing the matching content.

- TeaStory: Enables Kombucha content and uses TeaStory green tea fluid and black tea leaves.

- Enchantment Descriptions: Adds a description tooltip for the Algae Clearing enchantment.
